# Image assets

These files are NOT shipped by the template, drop the band's own art here after generation.

Required files at this directory's root:

| File                   | Size                  | Used for                                             |
|------------------------|-----------------------|------------------------------------------------------|
| `bandphoto.jpg`        | 1200×1200 or 1200×630 | OG/Twitter card image (Discord/Slack/Twitter embeds) |
| `bandlogo.png`         | as small as possible  | Logo shown next to the "Official Site" button        |
| `favicon.png`          | 32×32                 | Browser tab icon                                     |
| `apple-touch-icon.png` | 144×144               | iOS home-screen icon                                 |

Album covers go in `img/albums/`. 600×600 square is recommended
